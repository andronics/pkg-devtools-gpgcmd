# GPG Command Wrapper - Encryption and key management

**Package**: `@devtools/gpgcmd`
**Version**: 0.1.0
**Repository**: `pkg-devtools-gpgcmd`

## Overview

GPG Command Wrapper - Encryption and key management

## Installation

### Prerequisites

This package requires the following external commands:

- `gpg`
- `gpg2`

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @devtools/gpgcmd

# Or install from local source
cd ~/.pkgs
stow gpgcmd
```

## Usage

### Development Tool

GPG Command Wrapper - Encryption and key management


## Configuration

Configuration files are typically stored in:
- `~/.config/gpgcmd/` - User configuration
- `~/.local/share/gpgcmd/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @devtools/gpgcmd

# Or manual unstow
cd ~/.pkgs
stow -D gpgcmd
```
